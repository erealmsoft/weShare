基础选择器

example
  1. 通配符

    匹配当前网页上的所有标签,权重最低

    *{property:value}
  2. 标签选择器

    匹配当前网页上面所有此种名称的标签，权重为1

    tagName{property:value}
  3. 类选择器

    匹配当前网页上面所有带有此class的标签，权重为10

    .class{property:value}
  4. ID选择器

    匹配当前网页上面带有此ID的标签，权重为100

    #id{property:value}
  5. 属性选择器

    匹配当前网页上面某个属性等于某一值的所有标签

    [attr='value']{property:value}

练习

小练习1：请判断以下几个span标签最终的字体颜色是什么

html

            <div class="selector-test">
                <span id="span_test1" style="background-color: #d5d5d5;" class="span-test">span1</span>
                <span id="span_test2" class="span-test">span2</span>
                <span id="span_test3" class="span-test">span3</span>
            </div>
css

                .selector-test .selector-span-test{
                    background-color: blue;
                }
                     .selector-test .span-test {
                    background-color: blue;
                }

                .selector-span-test #selector_span_test1 {
                    background-color: green;
                }

                #span_test1 {
                    background-color: yellow;
                }

                .selector-span-test {
                    background-color: #c0a16b !important;
                }
                .selector-span-test1 {
                    background-color:greenyellow;
                }
                .selector-span-test1 {
                    background-color:saddlebrown;
                }

result

span1 span2 span3 span3 span4

小练习2：写一个静态页面以下为要求

  1. 页面结构完整
  2. 语义明确
  3. 用链接的方式引入外部样式表
  4. 至少使用4种选择器
  5. 选择器中定义的样式规则无要求，但请尽量保持页面美观
