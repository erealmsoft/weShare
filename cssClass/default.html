什么是CSS?

层叠样式表 (Cascading Style Sheets),常缩写为 CSS， 是一门 样式表 (stylesheet) 语言，用来描述 HTML、XML（包括各种 XML 语言如 SVG、XHTML）文档的呈现。CSS 描述了在屏幕、电子纸、音频或其它媒体上如何渲染元素。

怎么使用CSS?

  1. 行内

    写在页面上的标签里面

    不推荐这样使用，可复用性差，可维护性差

  2. 嵌入式CSS

    写在html页面中

    尽量不要写内联样式，如果实在没办法，需要这样写的话就写在head里面的style标签里

  3. 外部链接

    样式写在单独的文件里面，文件后缀名为.css

    多个页面可以复用，方便维护

  4. 导入

    写在单独文件中

    不推荐使用，会破坏浏览器的并行加载

Syntax

selectorlist { property: value; [more property:value; pairs] }
